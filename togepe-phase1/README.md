# Pinitup

Pinitup is a full-stack web application currently under development.

## Tech Stack

- React
- Vite
- Node.js
- Express.js
- MongoDB

## Project Structure

```
client/
server/
docs/
```

## Getting Started

### Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### Run the project

Frontend

```bash
cd client
npm run dev
```

Backend

```bash
cd server
npm run dev
```
